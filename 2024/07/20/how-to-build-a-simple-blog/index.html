如何搭建一个简单博客

发表于 2024-07-20 分类于 博客 347 1 分钟

今天开始将一些本地的笔记搬到线上来，随即，搭建了这样的一个简单能用的博客

简单介绍

使用了以下东西：

  • Hexo：简单，使用人数多的博客框架
  • Github：笔记文件的版本控制
  • Github Page：静态文件托管
  • Github Action：源文件与静态文件为两个仓库，Push 到源文件仓库后，自动触发，部署到静态文件仓库
  • Cloudflare：域名管理、URL缩短、防盗链实现、CDN 加速等
  • BackBlaze：成本可忽略不计的对象存储服务
  • SpaceShip：域名购买
