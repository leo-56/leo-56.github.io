axios 是什么？

axios 是一个基于 promise 的 HTTP 库，可以用在浏览器和 node.js 中

axios 中文文档：axios中文文档|axios中文网 | axios

axios 可以用来干什么？

可以从 node.js 中创建 HTTP 请求

也可以拦截请求和响应
