JavaScript中的对象拷贝

发表于 2023-02-13 分类于 前端

今天在开发一个简单的功能的时候，出现了这样的问题

出现这种情况让我联想到了 Java 中的引用类型，赋值的时候只是赋予了对象的引用，并不是赋予了一个和当前变量对应的对象一样的对象

查看代码后，发现果然是之前将 record 直接赋给了电子书表单数据绑定的 ebook 变量

这也就印证了我之前的猜测，涉及到了这一部分的知识：对象引用和复制 | 现代 JavaScript 教程

在教程中提供了一种方法来进行对象的真正复制

Object.assign({}, user) 可以将 user 对象的副本赋值给 clone 变量，此时 user 与 clone 分别指向两个独立的对象
