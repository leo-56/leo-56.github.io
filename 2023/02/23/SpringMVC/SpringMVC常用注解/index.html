SpringMVC常用注解

发表于 2023-02-23 分类于 SpringMVC

SpringMVC 101 第一篇

1 @Controller

标注控制器以处理 DispatcherServlet 分发的请求

2 @ResponseBody

用于将 Controller 方法返回的对象，根据请求头中 Accept 的内容，通过适当的 HttpMessageConverter 转换为指定格式后，写入 Response 对象的 body 数据区

😀 @Controller + @ResponseBody = @RestController

3 @RequestMapping

用于处理请求地址映射，可以用在类上，也可以用在方法上

在这个注解中可以设置请求方法：GET、POST、PUT、DELETE

但是我一般会使用下列注解代替：

  • @PostMapping（增）
  • @DeleteMapping（删）
  • @PutMapping（改）
  • @GetMapping（查）
