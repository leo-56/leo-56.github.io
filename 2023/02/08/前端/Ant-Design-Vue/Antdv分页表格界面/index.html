表格子组件属性理解

  • columns：表示表格列的配置描述，这里填写的是后面定义的一个数组，数组中的每一个对象表示一列
  • dataSource: 填写数据数组名。这里填的是 ebooks，表示用 ebooks 中的数据来填充列表中的数据
  • pagination: 使用分页栏
  • rowKey：为每一行指定一个 key，这里指定的是 id
  • change：这个不是属性，是事件，所以它是用 v-on 来绑定的。它所绑定的函数在发生分页、排序、筛选变化时触发

表格列配置属性理解

  • title：列头显示文字。这个就是表头显示的值
  • dataIndex：列数据在数据项中对应的 key。在这里的话，绑定的就是 ebooks 数组中对象的键值(key)
  • key：Vue 需要的 key。Vue 需要唯一的 key
  • slots：可用于配置 slot。在这里的话就是用 customRender 来将具名插槽 cover 渲染到 <a-table> 子组件中
